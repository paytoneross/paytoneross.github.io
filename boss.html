Boss Fight Mini Game

       This project was a quick gameplay demo I put together over the course of a long weekend to get some practice with good programming practices and coding more complex game mechanics. This was also my introduction to AI state machine programming. Using free assets from the Unity asset store, I created an FPS boss fight with complex puzzle mechanics that the player must solve in order to defeat the boss.

       I learned a lot about AI state machines that I ended up applying to later projects, such as critter, and I also strengthened my skills and understanding of OOP and C#. This was also my first swing at creating 3D FPS gameplay, which I found to be a lot of fun.
